1. Descargar el proyecto desde GitHub
    Descargar ZIP
    Ir al repositorio en GitHub

    Extraer el ZIP en una carpeta

2. Instalar y ejecutar el Backend (Spring Boot)

    Abrir la carpeta:
    cd backend

    Ejecutar el backend:
    mvn spring-boot:run

    El backend se ejecuta en:
    http://localhost:8080

3. Instalar y ejecutar el Frontend (React)

    Abrir la carpeta:
    cd frontend

    Instalar dependencias:
    npm install


    Ejecutar el proyecto:
    npm run dev


    El frontend se ejecuta en:
    http://localhost:3000
